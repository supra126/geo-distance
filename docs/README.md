# @simoko/geo-distance — Docs & Playground

Documentation site and interactive playground for [@simoko/geo-distance](https://www.npmjs.com/package/@simoko/geo-distance).

Built with [Astro](https://astro.build). Deployed to [GitHub Pages](https://supra126.github.io/geo-distance/).

## Development

```bash
npm install
npm run dev
```

Open http://localhost:4321

## Build

```bash
npm run build
npm run preview
```

## Deployment

Automatically deployed via GitHub Actions when changes are pushed to `docs/` on the `main` branch.
